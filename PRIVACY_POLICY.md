## PRIVACY POLICY

### 1. Data Collection and Local Storage
We are committed to protecting your privacy with a "Local First" philosophy.

*   **No Personal Data Uploads**: The Application does not collect, store, or transmit any personally identifiable information (PII) to external servers.
*   **Local-Only Storage**: Any data you create (such as "Saved Tickets" or "Favorite Numbers") is stored **exclusively on your local device**. This data is never synced to any cloud service. Uninstalling the Application will permanently delete this data.
*   **Absolute Security**: Your saved data remains under your complete control and cannot be accessed by the developer or any third party.

### 2. Device Permissions
The Application requests specific permissions for core features only.

| Permission | Purpose | Data Handling |
|---|---|---|
| **Camera** | Used **solely for scanning QR codes** on official lottery tickets to quickly check results. | No photos or videos are taken, stored, or transmitted. Only the QR code string is processed locally. |
| **Notifications** | Used **exclusively for lottery draw result alerts**. | No advertising notifications. Users can disable this in system settings at any time. |
| **Internet** | Required to fetch the latest lottery results and display advertisements. | No personal data is transmitted. |

### 3. Third-Party Services
To provide a free service, the Application integrates the following third-party SDKs:

*   **Google AdMob**: Displays advertisements. May use anonymized device identifiers for ad personalization. See [Google's Privacy Policy](https://policies.google.com/privacy).
*   **Firebase Analytics**: Collects anonymous, aggregated usage statistics (e.g., screen views) to improve the Application. Does not identify individual users.
*   **Firebase Cloud Messaging (FCM)**: Delivers push notifications for lottery results.

### 4. Data Security
All network communications use industry-standard encryption (HTTPS/TLS). No sensitive personal information is ever collected or stored by the Application.
